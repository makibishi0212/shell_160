```
type コマンド => ビルトインか調べる
```

```
1    標準出力
2    標準エラー出力
2>&1 標準エラー出力を標準出力に向ける
|&   標準出力、標準エラー出力の両方を次のコマンドに渡す

```

```
echo $? => 最後に実行されたコマンドのステータス
```

### ブレース展開

```
echo {a,b}{c,d} => ac ad bc bd
echo {a,b}.{c,d} => a.c a.d b.c b.d
echo {2..10..2} => 2 4 6 8 10
```

### ファイルグロブ

```
ls ?.txt => 任意の1文字.txt
ls {1,2}?.txt => 1? 2?のtxt
ls [^29].txt  => 2,9以外の1文字.txt
ls **/file_? => 任意の階層にあるfile_?に適合するファイル
```
